# **Getting Started**

Blocks is a growing library of HTML website components and page builders. 

HTML website components

Each component is self-contained, uses semantic HTML, and includes its own styles (embedded in a `<style>` tag). You can copy any component and paste it directly into your project.

### **How to use the component library:**

1. Browse the component categories.  
2. Customize colors, text, and images as needed.  
3. Click the “Copy Code” button next to any component.  
4. Paste the code into your HTML file where you want the component to appear.

## **Page Builder**
